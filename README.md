# HighLevel AI Lead Bridge

A production-minded FastAPI integration for processing HighLevel lead events.

This is a self-directed portfolio implementation using a fictional business scenario and test data. It is not affiliated with, endorsed by, or officially connected to HighLevel.

## Development

Copy `.env.example` to `.env`, install the project, then run:

```powershell
uvicorn app.main:app --reload
```

The service exposes `GET /health` and `GET /ready`.

Further setup and integration documentation will be added as the project is built.
